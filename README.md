## Run Locally

Clone the project

```bash
  git clone https://github.com/ei-abdullah/far-away.git
```

Go to the far-away directory

```bash
  cd far-away
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

**Client:** React, Javascript, CSS, HMTL
